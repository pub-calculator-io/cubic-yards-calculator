Cubic yard calculator calculates cubic yards, feet, meters for landscape materials. Enter measurements in yards and feet and get volume conversions.

[https://www.calculator.io/cubic-yards-calculator/](https://www.calculator.io/cubic-yards-calculator/)

== Usage ==

Add the Cubic Yards Calculator shortcode to your page, post or sidebar:

`[ci_cubic_yards_calculator]`

Add the following code to your template where you would like the Cubic Yards Calculator to appear:

`<?php display_ci_cubic_yards_calculator(); ?>`

2. Activate the Cubic Yards Calculator plugin through the "Plugins" menu in WordPress.
